# PuzzleFree-Button-Firefox

A lightweight browser extension that adds quick access to https://puzzlefree.game.

## Features
- One-click access from the browser toolbar
- No tracking
- No data collection
- No special permissions

## Build
No build process required.

## Source code
This repository contains the exact source code submitted to the browser extension store.

## Addon
https://addons.mozilla.org/en-US/firefox/addon/puzzlefree-button/
